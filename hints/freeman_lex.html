I still need a hint.

When you've found a new occurrence of substring and ind have got a new value, you need to increment your res variable's value and you also need to increment ind by 1 to start a new search from the next character after previous finding. 

res += 1
ind += 1
So, how my final solution should look like?

You just need to join all previous hints, return the result and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def count_occurrences(main_str: str, sub_str: str) -> int:

    ind = res = 0
    main_str = main_str.lower()
    sub_str = sub_str.lower()
    while (ind := main_str.find(sub_str, ind)) != -1:
        res += 1
        ind += 1
        
    return res

Using regular expressions find all occurrences without consuming characters (overlapping allowed) and ignoring case. 

import re


def count_occurrences(main_str: str, sub_str: str) -> int:
    
    matches = re.findall(fr'(?={sub_str})', main_str, flags=re.IGNORECASE)
    
    return len(matches)
