Ich brauche noch einen Hinweis.

Wenn Sie ein neues Vorkommen einer Teilzeichenkette gefunden haben und ind einen neuen Wert erhalten hat, müssen Sie den Wert Ihrer Variable res erhöhen und auch ind um 1 erhöhen, um eine neue Suche ab dem nächsten Zeichen nach dem vorherigen Fund zu starten. 

res += 1
ind += 1
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur alle vorherigen Hinweise kombinieren, das Ergebnis zurückgeben und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def count_occurrences(main_str: str, sub_str: str) -> int:

    ind = res = 0
    main_str = main_str.lower()
    sub_str = sub_str.lower()
    while (ind:= main_str.find(sub_str, ind))!= -1:
        res += 1
        ind += 1
        
    return res

Verwenden Sie reguläre Ausdrücke, um alle Vorkommen zu finden, ohne Zeichen zu verbrauchen (Überschneidungen erlaubt) und ohne die Groß- und Kleinschreibung zu beachten. 

import re


def count_occurrences(main_str: str, sub_str: str) -> int:
    
    matches = re.findall(fr'(?={sub_str})', main_str, flags=re.IGNORECASE)
    
    return len(matches)
