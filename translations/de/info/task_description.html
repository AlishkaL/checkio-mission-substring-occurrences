Diese Funktion sollte eine Hauptzeichenkette und eine Teilzeichenkette als Eingaben annehmen und die Anzahl der Vorkommen der Teilzeichenkette in der Hauptzeichenkette zurückgeben. Sie sollte Groß- und Kleinschreibung nicht berücksichtigen und kann sich überschneiden.

Eingabe: Zwei Zeichenketten (str{% if is_js %}ing{% endif %}).

Ausgabe: Ganzzahl ({% if is_js %}number{% else %}int{% endif %}).

{{init_code_tmpl}}

Wie es verwendet wird:

  • in einer Textverarbeitungssoftware, um alle Vorkommen eines bestimmten Wortes oder Ausdrucks zu markieren oder zu ersetzen;
  • beim Scraping von Websites, um zu zählen, wie oft ein bestimmtes Schlüsselwort vorkommt;
  • in einer Analyseanwendung, um die Häufigkeit bestimmter Wörter oder Ausdrücke in einem Text zu analysieren.

Vorbedingungen:

  • beide Eingaben sollten Strings sein: mainStr ∈ String und subStr ∈ String;
  • die Hauptzeichenkette kann eine beliebige Länge haben, einschließlich Null: 0 <= length(mainStr) <= N;
  • die Teilzeichenkette sollte keine leere Zeichenkette sein: length(subStr) > 0.
