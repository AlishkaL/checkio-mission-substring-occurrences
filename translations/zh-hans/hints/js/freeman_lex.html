为子串的起始索引和出现次数声明变量，初始值为 0。

将给定的两个字符串转换为相同大小写，以捕捉子串的所有可能变体。

注意，这一步很重要。你需要从当前的index 开始，在主字符串中不断查找子串。使用string.indexOf() 方法，用表达式的结果更新变量值，在主字符串和while loop 上 "移动"。 

while ((index = mainStr.indexOf(subStr, index))!== -1) {
    ...
}

重复上述操作，直到从当前index 开始的主字符串中出现子字符串。

我还需要提示。

当找到一个新的子串且index 有了新值时，需要将count 变量的值递增，同时将index 的值递增 1，以便从上一次搜索后的下一个字符开始新的搜索。 

count++;
index++;
那么，我的最终解决方案应该是怎样的呢？

您只需将之前的所有提示连接起来，返回结果并将代码封装成函数表达式。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

function countOccurrences(mainStr: string, subStr: string): number {

    let count = 0, index = 0;
    mainStr = mainStr.toLowerCase();
    subStr = subStr.toLowerCase();
    while ((index = mainStr.indexOf(subStr, index))!== -1) {
        count++;
        index++;
    }
    return count;
}

使用正则表达式查找所有出现的字符，不消耗字符（允许重叠），忽略大小写。 

function countOccurrences(mainStr: string, subStr: string): number {

    let matches = mainStr.match(new RegExp(`(?=${subStr})`, "gi"));
    
    return matches? matches.length: 0;
}
