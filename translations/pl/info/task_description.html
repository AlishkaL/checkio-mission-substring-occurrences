Ta funkcja powinna przyjmować główny ciąg znaków i podciąg jako dane wejściowe i zwracać liczbę wystąpień podciągu w głównym ciągu znaków. Nie powinna rozróżniać wielkości liter i może się nakładać.

Wejście: Dwa ciągi znaków (str{% if is_js %}ing{% endif %}).

Wyjście: Liczba całkowita ({% if is_js %}number{% else %}int{% endif %}).

Przykłady: 

{{init_code_tmpl}}

Jak to jest używane:

  • w edytorze tekstu do podświetlania lub zastępowania wszystkich wystąpień określonego słowa lub frazy;
  • w skrobaniu stron internetowych w celu zliczenia liczby wystąpień określonego słowa kluczowego;
  • w aplikacji analitycznej do analizy częstotliwości występowania określonych słów lub fraz w tekście.

Warunki wstępne:

  • oba dane wejściowe powinny być ciągami znaków: mainStr ∈ string i subStr ∈ string;
  • główny ciąg może mieć dowolną długość, w tym zero: 0 <= length(mainStr) <= N;
  • podłańcuch nie powinien być pustym łańcuchem: length(subStr) > 0.
