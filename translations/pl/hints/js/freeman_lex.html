Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę zadeklarować zmienne dla indeksu początkowego podciągu i liczby wystąpień, przekonwertować oba podane ciągi na tę samą wielkość liter i znaleźć podciąg w głównym ciągu, zaczynając od bieżącego indeksu początkowego, gdy jest tam obecny, zaktualizować indeks początkowy po znalezieniu.

Ok, od czego powinienem zacząć?

Proszę zadeklarować zmienne dla indeksu początkowego podciągu i liczby wystąpień z wartościami początkowymi 0. 

let count = 0, index = 0;

Proszę przekonwertować oba podane ciągi na tę samą wielkość liter, aby wychwycić wszystkie możliwe warianty podciągu. 

mainStr = mainStr.toLowerCase();
subStr = subStr.toLowerCase();
Co robić dalej?

Proszę uważać, to ważny krok. Należy stale szukać podciągów w głównym ciągu, zaczynając od bieżącego index. Proszę użyć metody string.indexOf() i zaktualizować wartość zmiennej wynikiem wyrażenia, aby "przesunąć się" po głównym ciągu i while loop. 

while ((index = mainStr.indexOf(subStr, index))!== -1) {
    ...
}

Proszę powtarzać, aż podciąg będzie obecny w głównym ciągu, zaczynając od bieżącego index.

Nadal potrzebuję podpowiedzi.

Po znalezieniu nowego wystąpienia podciągu i index ma nową wartość, należy zwiększyć wartość zmiennej count, a także zwiększyć index o 1, aby rozpocząć nowe wyszukiwanie od następnego znaku po poprzednim znalezieniu. 

count++;
index++;
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi, zwrócić wynik i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

function countOccurrences(mainStr: string, subStr: string): number {

    let count = 0, index = 0;
    mainStr = mainStr.toLowerCase();
    subStr = subStr.toLowerCase();
    while ((index = mainStr.indexOf(subStr, index))!== -1) {
        count++;
        index++;
    }
    return count;
}

Jasne, proszę pobrać!

Za pomocą wyrażeń regularnych proszę znaleźć wszystkie wystąpienia bez zużywania znaków (nakładanie się dozwolone) i ignorowania wielkości liter. 

function countOccurrences(mainStr: string, subStr: string): number {

    let matches = mainStr.match(new RegExp(`(?=${subStr})`, "gi"));
    
    return matches? matches.length: 0;
}
