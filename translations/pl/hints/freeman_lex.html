Proszę zadeklarować zmienne dla indeksu początkowego podciągu i liczby wystąpień, przekonwertować oba podane ciągi na tę samą wielkość liter i znaleźć podciąg w głównym ciągu, zaczynając od bieżącego indeksu początkowego, gdy jest tam obecny, zaktualizować indeks początkowy po znalezieniu.

Ok, od czego powinienem zacząć?

Proszę zadeklarować zmienne dla indeksu początkowego podciągu i liczby wystąpień z wartościami początkowymi 0. 

ind = res = 0

Proszę przekonwertować oba podane ciągi na tę samą wielkość liter, aby wychwycić wszystkie możliwe warianty podciągu. 

main_str = main_str.lower()
sub_str = sub_str.lower()
Co robić dalej?

Proszę uważać, to ważny krok. Należy stale szukać podciągów w głównym ciągu, zaczynając od bieżącego ind. Proszę użyć metody str.find() i zaktualizować wartość zmiennej wynikiem wyrażenia, aby "przesunąć się" po głównym ciągu. Proszę użyć operatora walrus i while loop. 

while (ind:= main_str.find(sub_str, ind))!= -1:
    ...

Proszę powtarzać, aż podciąg będzie obecny w głównym ciągu, zaczynając od bieżącego ind.

Nadal potrzebuję podpowiedzi.

Po znalezieniu nowego wystąpienia podciągu i ind ma nową wartość, należy zwiększyć wartość zmiennej res, a także zwiększyć ind o 1, aby rozpocząć nowe wyszukiwanie od następnego znaku po poprzednim znalezieniu. 

res += 1
ind += 1
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi, zwrócić wynik i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

def count_occurrences(main_str: str, sub_str: str) -> int:

    ind = res = 0
    main_str = main_str.lower()
    sub_str = sub_str.lower()
    while (ind:= main_str.find(sub_str, ind))!= -1:
        res += 1
        ind += 1
        
    return res

Jasne, proszę łapać!

Za pomocą wyrażeń regularnych proszę znaleźć wszystkie wystąpienia bez zużywania znaków (nakładanie się dozwolone) i ignorując wielkość liter. 

import re


def count_occurrences(main_str: str, sub_str: str) -> int:
    
    matches = re.findall(fr'(?={sub_str})', main_str, flags=re.IGNORECASE)
    
    return len(matches)
